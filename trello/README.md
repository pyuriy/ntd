## Python script щоб отримати перелік карток з приватної дошки Trello

Оскільки Trello має відкрите REST API, ви можете завантажити та запустити скрипт на Python, щоб отримати список карток з будь-якої дошки (включаючи приватні, якщо у вас є відповідний токен доступу).

Ось як це реалізувати технічно:

### 1. Що знадобиться на iPhone

Щоб запускати Python-скрипти безпосередньо на iOS, вам знадобиться спеціальний застосунок. Найкраще для цього підходить **Pythonista 3** або безкоштовний аналог **a-Shell** (емулятор терміналу з підтримкою Python).

### 2. Отримання доступів до Trello API

Оскільки дошка приватна, скрипту потрібні ваші облікові дані для авторизації.

1. Перейдіть на сторінку [Trello Power-Up Admin Central](https://trello.com/power-ups/admin).
2. Створіть новий New Power-Up (це формальність для отримання ключів).
3. Скопіюйте ваш **API Key**.
4. Поруч за посиланням згенеруйте персональний **Token** (він дасть скрипту доступ до ваших приватних дошок).
5. Знайдіть **Board ID** вашої дошки (його можна взяти з URL-адреси дошки в браузері: `https://trello.com/b/[це_ваш_board_id]/назва-дошки`).

---

### 3. Скрипт на Python

Вам знадобиться бібліотека `requests`. У Pythonista вона вже вбудована. Якщо використовуєте a-Shell, її можна встановити командою `pip install requests`.

Ось готовий код скрипту:

```python
import requests

# Вкажіть ваші дані Trello
API_KEY = 'ВАШ_API_KEY'
TOKEN = 'ВАШ_TOKEN'
BOARD_ID = 'ID_ВАШОЇ_ДОШКИ'

# URL для отримання карток з конкретної дошки
url = f"https://api.trello.com/1/boards/{BOARD_ID}/cards"

query = {
    'key': API_KEY,
    'token': TOKEN
}

try:
    response = requests.request("GET", url, params=query)
    
    if response.status_code == 200:
        cards = response.json()
        print(f"--- Знайдено карток: {len(cards)} ---\n")
        
        for index, card in enumerate(cards, 1):
            print(f"{index}. {card['name']}")
            if card['desc']:
                print(f"   Опис: {card['desc'][:50]}...") # перші 50 символів опису
    else:
        print(f"Помилка запиту: {response.status_code}")
        print(response.text)

except Exception as e:
    print(f"Сталася помилка: {e}")

```

### Альтернативний варіант (без Python)

Якщо мета — просто автоматизувати отримання карток на iPhone без прив'язки до коду, це можна зробити через вбудований застосунок **Apple Shortcuts (Команди)** за допомогою дії «Отримати вміст URL-адреси» (Get contents of URL). Це працюватиме швидше і не потребуватиме стороннього софту для запуску Python.

### Покроковий шлях, як дійти до генерації токена:

1. Перейдіть за посиланням вище та увійдіть під своїм акаунтом Trello.
2. Натисніть кнопку **New Power-Up** (або *Створити нове розширення*).
3. Заповніть обов'язкові поля (можна вказати будь-яку назву, наприклад `MyPythonScript`, вибрати свій робочий простір та натиснути **Create**).
4. Після створення розширення перейдіть на вкладку **API Key** (зазвичай вона з'являється в меню ліворуч).
5. Натисніть кнопку **Generate API Key**.
6. Перед вами з'явиться ваш **API Key**, а прямо під ним (або поруч у тексті) буде фраза:
> *"You can manually generate a **Token** to access your own account..."* Слово **Token** у цьому реченні буде клікабельним посиланням. Натисніть на нього, дозвольте доступ до свого акаунту, і система видасть вам фінальний довгий рядок символів — це і є ваш секретний токен для скрипту.

## Токен тільки з дозволом читати?

Так, Trello API дозволяє це зробити. Під час генерації токена ви можете чітко вказати обмеження: дозволити лише читання (**read**) та заборонити запис або зміну даних (**write**).

Для цього потрібно вручну сформувати правильне посилання для авторизації, додавши туди параметр `scope=read`.

### Як згенерувати токен "тільки для читання"

Замініть `ВАШ_API_KEY` у посиланні нижче на ваш реальний ключ, отриманий на попередньому кроці, та перейдіть за цією адресою у браузері:

`https://trello.com/1/authorize?expiration=never&scope=read&response_type=token&key=ВАШ_API_KEY`

---

### Що означають ці параметри в URL?

* `scope=read` — **найголовніший параметр**. Він вказує, що скрипт зможе лише читати дошки, списки та картки, але будь-яка спроба щось створити, видалити чи відредагувати завершиться помилкою `401 Unauthorized`.
* `expiration=never` — токен буде вічним, поки ви самі не відкличете його в налаштуваннях профілю Trello. Якщо безпека понад усе, можна вказати `expiration=30days` (дійсний 30 днів) або `expiration=1day` (на 1 день).
* `response_type=token` — каже Trello повернути готовий токен прямо в браузер.

Коли ви перейдете за цим посиланням, Trello запитає у вас дозвіл. Зверніть увагу на текст запиту: там буде чітко вказано, що застосунок отримає доступ **"read your boards"** (читати ваші дошки), але не буде пунктів про зміну чи видалення даних. Натисніть **Allow** (Дозволити), і на екрані з'явиться ваш безпечний токен.


### Як дізнатися повний (довгий) ID дошки?

Якщо вам або вашій бібліотеці колись знадобиться повний унікальний 24-значний ID цієї дошки, є два простих способи його знайти:

**Спосіб 1: Через зміну URL (найшвидший)**

1. Відкрийте ваше посилання у браузері.
2. Додайте `.json` в кінець адреси після назви дошки.
3. Посилання виглядатиме так: `https://trello.com/b/7E40mWdc/welcome-board.json`
4. Перейдіть за ним. Сторінка завантажиться у вигляді тексту (JSON-коду). Найперший параметр у цьому тексті — `"id"` (наприклад, `"id": "60b9f...""`). Це і є повний ID.

**Спосіб 2: Через розширене меню Trello**

1. Відкрийте дошку в Trello.
2. У правому верхньому кутку натисніть **Меню** (три крапки `...`).
3. Виберіть пункт **Ще** (More), а потім **Друк та експорт** (Print and export).
4. Натисніть **Експорт у JSON** (Export as JSON).
5. На екрані відкриється той самий текстовий код, де першим значенням буде `"id"`.

## Перелік списків на дошці?

Щоб отримати перелік **списків** (Lists/Колонок) на дошці, вам потрібно змінити кінцеву точку (endpoint) в URL-адресі запиту. Замість `/cards` ми використовуємо `/lists`.

Ось як буде виглядати оновлений Python-скрипт. Я додав виведення унікального ID для кожного списку, оскільки він часто потрібен для подальшої роботи (наприклад, щоб дізнатися, які саме картки лежать у конкретній колонці).

### Оновлений скрипт

```python
import requests

# Ваші дані Trello
API_KEY = 'ВАШ_API_KEY'
TOKEN = 'ВАШ_TOKEN'
BOARD_ID = '7E40mWdc'  # Ваш ID дошки

# Змінений URL: тепер ми звертаємося до /lists
url = f"https://api.trello.com/1/boards/{BOARD_ID}/lists"

query = {
    'key': API_KEY,
    'token': TOKEN
}

try:
    response = requests.request("GET", url, params=query)
    
    if response.status_code == 200:
        lists = response.json()
        print(f"--- Знайдено списків (колонок): {len(lists)} ---\n")
        
        for index, t_list in enumerate(lists, 1):
            # Виводимо назву списку та його ID
            print(f"{index}. Назва: \"{t_list['name']}\"")
            print(f"   ID списку: {t_list['id']}\n")
    else:
        print(f"Помилка запиту: {response.status_code}")
        print(response.text)

except Exception as e:
    print(f"Сталася помилка: {e}")

```

### Що змінилося?

1. **URL:** Тепер запит іде на `https://api.trello.com/1/boards/{BOARD_ID}/lists`.
2. **Результат:** Скрипт поверне масив об'єктів, де кожен об'єкт — це колонка на вашій дошці (наприклад: *To Do, In Progress, Done*), відсортована в тому порядку, в якому вони розташовані на екрані.
